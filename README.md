# 🌐 SocialConnect

O **SocialConnect** é um site institucional desenvolvido com o objetivo de simular a presença digital de uma empresa, aplicando boas práticas de versionamento com Git e GitHub.

Durante o desenvolvimento, foram corrigidos problemas existentes no projeto e adicionadas novas funcionalidades, utilizando uma estratégia de branching e colaboração em equipe.

---

## Funcionalidades

* ✅ Correção de imagens quebradas
* ✅ Padronização de cores (#2980b9)
* ✅ Atualização do rodapé (ano e textos)
* ✅ Correção da seção "Sobre Nós"
* ✅ Adição da seção de equipe
* ✅ Inclusão do item "Projetos" no menu
* ✅ Criação da seção de contato
* ✅ Adição de redes sociais no rodapé
